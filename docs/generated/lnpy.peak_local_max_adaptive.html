lnpy.peak_local_max_adaptiveÔÉÅ

lnpy.peak_local_max_adaptive(data, mask=None, min_distance=None, threshold_rel=0.0, threshold_abs=0.2, num_peaks_max=None, style='indices', connectivity=None, errors='warn', **kwargs)[source]ÔÉÅ

Find local max with fall backs min_distance and filter.

This is an adaptation of peak_local_max(), which is called iteratively until the number of peaks is less than num_peaks_max.

Parameters

  • data (array-like) ‚Äì Image data to analyze

  • mask (bool or array-like, optional) ‚Äì This uses the ‚Äúimage‚Äù convention. Where mask is True, values are include.

  • min_distance (int or sequence of ints, default (5, 10, 15, 20, 25)) ‚Äì min_distance parameter. If sequence, then call peak_local_max() until number of peaks <=num_peaks_max.

  • threshold_rel, threshold_abs (float) ‚Äì thresholds parameters

  • num_peaks_max (int, optional.) ‚Äì Max number of maxima/peaks to find. If not specified, any number of peaks allowed.

  • style ({'indices', 'mask', 'marker'}) ‚Äì Controls output style

    • indices : indices of peaks

    • mask : bool array marking peak locations

    • marker : array of ints

  • connectivity (int, optional) ‚Äì Maximum number of orthogonal hops to consider a pixel/voxel as a neighbor. Accepted values are ranging from 1 to input.ndim. If None, a full connectivity of input.ndim is used.

  • indices (bool, optional, default True) ‚Äì if True, return indicies of peaks. if False, return array of ints of shape data.shape with peaks marked by value > 0.

  • errors ({'ignore','raise','warn'}, default 'warn') ‚Äì

    • If raise, raise exception if npeaks > num_peaks_max

    • If ignore, return all found maxima

    • If warn, raise warning if npeaks > num_peaks_max

  • **kwargs ‚Äì Extra arguments to peak_local_max()

Returns

out ‚Äì Depending on the value of indices.

Return type

indices or mask

Notes

The option mask is passed as the value labels in peak_local_max()
