lnpy.merge_regionsÔÉÅ

lnpy.merge_regions(w_tran, w_min, masks, nfeature_max=None, efac=1.0, force=True, convention='image', warn=True)[source]ÔÉÅ

Merge labels where free energy energy barrier < efac.

The scaled free energy \(w = \beta f = - \ln \Pi \)Delta w = text{w_tran} - text{w_min} < text{efac}` will be merged into a single phase. Here, w_trans is the transition energy between phases (i.e., the minimum value of lnPi` between regions`) and w_min is the minimum energy (the maximum lnPi) for a phase.

Parameters

  • w_tran (array) ‚Äì Shape of array is (n, n), where n is the number of unique regions/phases.

  • This is the transitional free energy ()

  • w_min (array) ‚Äì Shape of array is (n, 1).

  • masks (sequence of bool arrays) ‚Äì Masks for each region using convention

  • nfeature_max (int) ‚Äì maximum number of features/phases to allow

  • efac (float, default 0.5) ‚Äì Wnergy difference to merge on. When w_trans[i, j] - w_min[i] < efac, phases i and j will be merged together.

  • force (bool, default True) ‚Äì if True, then keep going until nfeature <= nfeature_max even if min_val > efac.

  • convention (string or bool) ‚Äì Convention for mask. Allowable values are:

    • ‚Äòimage‚Äô or True : True values included, False values excluded. This is the normal convention in scipy.ndimage.

    • ‚Äòmasked‚Äô or False: False values are included, True values are excluded. This is the convention in numpy.ma

  • warn (bool, default True) ‚Äì if True, give warning messages

Returns

  • masks (list of bool arrays) ‚Äì output masks using convention

  • w_trans (array) ‚Äì transition energy for new masks

  • w_min (array) ‚Äì free energy minima of new masks
